# 补虚药

>现代都市人大多在气血阴阳等某一方面有所不足,即有体虚表现,这时候需要合理服用补虚药,使用补虚药时,长将两类或两类以上的补虚药配伍使用.还要防止为补而补,避免不分体质,不分气血阴阳,不分寒热属性乱补,  
注意补虚配方中处理好袪邪和扶正的关系,另外,补虚药做汤剂,一般要久煎,使药性全部析出
